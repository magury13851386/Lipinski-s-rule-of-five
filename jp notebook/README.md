# Lipinski Rule of Five – Data Analysis

## Goal
Analyze a dataset of compounds and classify them as Pass/Fail based on Lipinski's Rule of Five (Ro5).

## Ro5 Criteria
A compound is labeled as Pass if:
- MW ≤ 500
- LogP ≤ 5
- HBA ≤ 10
- HBD ≤ 5

## Key Results
- Total: 500 compounds
- Pass: 89 (~17.8%)
- Fail: 411 (~82.2%)

## Mean Statistics
- Pass: MW=363.42, LogP=2.15, HBA=4.78, HBD=2.23
- Fail: MW=533.92, LogP=3.29, HBA=7.62, HBD=3.90
